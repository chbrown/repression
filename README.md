## Installation

**From npm:**

```sh
npm install repression
```

**Or github:**

```sh
git clone https://github.com/chbrown/repression.git
cd repression
npm install -g
```

**Import:**

```js
var repression = require('repression');
```

      "postinstall": "... install"
